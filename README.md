# Hotel Reservation System

## 📖 About

Hotel Reservation System is a Java console-based application developed as part of the **CodeAlpha Java Programming Internship**. The application allows users to search available rooms, book hotel rooms, cancel reservations, and manage booking records. The project demonstrates the implementation of Object-Oriented Programming concepts and Java File Handling for storing reservation details.

---

## ✨ Features

- Search available rooms
- Book hotel rooms
- Cancel reservations
- View booking details
- Store booking records using file handling
- Console-based user interface

---

## 🛠 Technologies Used

- Java
- File Handling
- Object-Oriented Programming (OOP)
- Console-Based Application

---

## 📚 OOP Concepts Used

- Classes & Objects
- Encapsulation
- Methods
- Constructors

---

## ▶️ How to Run

1. Clone this repository.
2. Open the project in any Java IDE (VS Code, IntelliJ IDEA, Eclipse, etc.).
3. Compile all Java files.
4. Run `Main.java`.
5. Follow the on-screen instructions to search, book, and manage hotel reservations.
